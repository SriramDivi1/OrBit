# PRD: Sriram Divi Portfolio Website

## Problem Statement
Build a standout software developer portfolio website for Sriram Divi - Full Stack Developer. Features include About Me, Projects, Skills, Experience/Resume, Contact Form sections with dark/light mode toggle, downloadable resume, social links, and animations.

## User Personas
- **Recruiters/Hiring Managers**: Looking to evaluate developer skills, experience, and projects
- **Potential Clients**: Seeking to hire for freelance/contract work
- **Tech Community**: Fellow developers, collaborators

## Core Requirements
- Hero section with name, typewriter role effect, CTAs
- About section with terminal-style developer info
- Projects gallery with filter tabs (All, Frontend, Backend, DevOps)
- Skills section with progress bars and tech marquee
- Experience timeline with work history
- Contact form with backend API integration
- Dark/Light theme toggle
- Downloadable resume
- Social links (GitHub, LinkedIn, Email)
- Smooth animations and transitions

## Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion
- **Backend**: FastAPI, MongoDB
- **Fonts**: Clash Display, Manrope, JetBrains Mono
